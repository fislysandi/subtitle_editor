## Installation

### Prerequisites

- Blender 4.5 LTS or 5.0+
- ~500 MB disk space for dependencies
- (Optional) GPU with CUDA/ROCm/Metal support for faster transcription

### Install Steps

1. Download the latest release ZIP from the releases page
2. In Blender: `Edit → Preferences → Add-ons → Install...`
3. Select the ZIP file and click **Install Add-on**
4. Enable the checkbox next to **Subtitle Studio**
5. Open the **Add-on Preferences** and click **Install Dependencies**

```
Edit
└── Preferences
    └── Add-ons
        └── Install...
            └── Select subtitle_studio.zip
                └── Enable ✅ Subtitle Studio
                    └── Install Dependencies
```

## First-Time Setup

### Install Dependencies

After enabling the add-on:

1. Click the arrow to expand **Subtitle Studio** in the add-ons list
2. Click **Install Dependencies** button
3. Wait for installation to complete (may take 2-5 minutes)
4. Click **Install PyTorch** and select your GPU backend:
   - **CPU Only** - Works on all systems (slower)
   - **CUDA 12.1** - For RTX 30/40 series (recommended)
   - **Metal** - For Apple Silicon Macs

### Download a Whisper Model

1. Open the **Video Sequence Editor**
2. Press `N` to open the sidebar
3. Click the **Subtitle Studio** tab
4. In the **Whisper Model** section, select a model:
   - `base` - Fast, good accuracy (recommended for beginners)
   - `small` - Balanced speed/quality
   - `medium` - Better accuracy (requires GPU)
5. Click **Download Model**

## Basic Workflow

### 1. Add Media to VSE

```python
# Typical setup
Video Strip (Channel 1)     → Your footage
Audio Strip (Channel 1)     → Your audio
Subtitle Strips (Channel 2+) → Generated by Subtitle Studio
```

### 2. Select Audio/Video Strip

Click on the strip you want to transcribe in the sequencer.

### 3. Transcribe

1. Open the **Subtitle Studio** panel (`N` key)
2. Click the **Transcribe Audio** button
3. Wait for transcription to complete
4. Subtitle strips appear on the designated channel

### 4. Edit Subtitles

1. Select a subtitle from the list
2. Edit text in the **Current Text** field
3. Use **Nudge** buttons to adjust timing
4. Apply styles with preset buttons

## Transcription

### Model Selection

| Model | Size | Speed | Best For |
|-------|------|-------|----------|
| `tiny` | 39 MB | Very Fast | Testing, CPU-only |
| `base` | 74 MB | Fast | Quick drafts |
| `small` | 244 MB | Moderate | Balanced workflow |
| `medium` | 769 MB | Slow | High accuracy |
| `large-v3` | 1.5 GB | Very Slow | Best quality |

### Language Settings

- **Auto-detect**: Let Whisper detect the language automatically
- **Specific Language**: Choose from 99+ supported languages
- **Translate to English**: Convert any language to English subtitles

### Advanced Transcription Options

```python
# Access via the UI panel
Recognition Settings:
├── Language: auto / en / es / fr / ...
├── Beam Size: 1-10 (higher = more accurate, slower)
├── VAD Filter: On/Off (removes non-speech)
└── Advanced:
    ├── VAD Threshold: 0.0-1.0 (lower = more sensitive)
    ├── Min Speech Duration: 0-5000ms
    └── Min Silence Duration: 0-10000ms
```

### VAD Tuning for Music

For lyrics or speech with background music:

```
VAD Threshold: 0.3-0.4 (default: 0.5)
Min Speech Duration: 100-150ms (default: 250ms)
Min Silence Duration: 500-1000ms (default: 2000ms)
```

## Subtitle Editing

### List View Navigation

The subtitle list shows all text strips on your designated channels:

```
┌─────────────────────────────────────┐
│ ▓ Subtitle_001: "Hello World"      │
│   Subtitle_002: "Welcome back"     │
│   Subtitle_003: "Let's begin"      │
└─────────────────────────────────────┘
```

Click a subtitle to select it. The timeline jumps to the subtitle's start frame.

### Text Editing

1. Select a subtitle from the list
2. Type in the **Editing:** text field
3. Changes apply immediately to the strip

### Timing Adjustments

```
Editing: Subtitle_001
├─ Start: [  120] [<-] [->]    # Jump to Start, Nudge Left/Right
├─ End:   [  150] [<-] [->]    # Jump to End, Nudge Left/Right
└─ Speaker: [Speaker 1 ▼]

Nudge Step: 1 frame (adjust in settings)
```

### Frame Navigation

| Action | Button | Shortcut |
|--------|--------|----------|
| Jump to Start | Start | - |
| Jump to End | End | - |
| Select Next | ▲ | - |
| Select Previous | ▼ | - |

## Speaker Management

### Adding Speakers

1. Click **+** next to the Speaker dropdown to add a speaker
2. Enter the speaker name in the dialog
3. The speaker appears as a new channel

### Speaker Tabs

```
Speaker: [Speaker 1 ▼] [+] [-]
Channels: Ch 2: Speaker 1 | Ch 3: Speaker 2 | Ch 4: Speaker 3
```

Each speaker gets their own channel. Click **-** to remove a speaker.

### Speaker Assignment

1. Select a subtitle from the list
2. Choose a speaker from the **Speaker** dropdown
3. The subtitle moves to that speaker's channel
4. Text is prefixed with speaker name (optional)

### Speaker Prefix

Enable **Prefix in Text** to automatically add speaker names:

```
Before: "Hello everyone"
After:  "John: Hello everyone"
```

## Styling

### Style Presets

Three built-in presets for quick styling:

| Preset | Font Size | Use Case |
|--------|-----------|----------|
| Default | 24px | General subtitles |
| Lower Third | 28px | Professional videos |
| Large | 40px | Presentations |

### Style Properties

```python
# Available style options
Font Size: 8-200px
Text Color: RGB (0.0-1.0)
Shadow Color: RGB (0.0-1.0)
V Align: TOP / CENTER / BOTTOM
Wrap Width: 0.0-1.0 (percentage of screen)
Max Chars: 10-200 per line
```

### Apply Styles

1. Set style properties in the panel
2. Click **Copy Active Style to Selected** to apply to multiple strips
3. Or click **Save Style Preset** to save current settings

### Batch Operations

```
Batch Styling
└── [Copy Active Style to Selected]

# Select multiple strips in the sequencer
# Click the button to apply current style to all selected
```

## Import & Export

### Import Subtitles

Supported formats:
- **SRT** - SubRip (most common)
- **VTT** - WebVTT (web videos)
- **ASS** - Advanced SSA (styled subtitles)

1. Click **Import** button in the subtitle list
2. Select your subtitle file
3. Subtitles are placed on the designated channel

### Export Subtitles

1. Click **Export** button in the subtitle list
2. Choose location and format
3. All text strips are exported with timing preserved

### Format Auto-Detection

The add-on automatically detects format from file extension:

```python
.srt  → SubRip format
.vtt  → WebVTT format  
.ass  → Advanced SSA format
.ssa  → SubStation Alpha format
```

## Keyboard Shortcuts

Currently, Subtitle Studio uses Blender's built-in shortcuts:

| Shortcut | Action |
|----------|--------|
| `N` | Toggle sidebar panel |
| `Shift + ←/→` | Nudge frame left/right |
| `Ctrl + Z` | Undo |
| `Ctrl + Shift + Z` | Redo |

Panel-specific shortcuts:
- Click subtitle in list → Select + jump to frame
- Double-click speaker tab → Rename speaker
